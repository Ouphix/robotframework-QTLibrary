robotframework-QTLibrary
========================

QTLibrary是一个个人学习研究时做的测试库，参考了一下Selenium2Library的一些写法.
其中做了几个比较常用的关键字，比如随机生成生日，随机生成身份证号，随机生成中文姓名等。

如果不知道关键字如何使用，请查看一下test目录里的案例。

安装方法：

>pip install robotframework-qtlibrary

Installation :

>pip install robotframework-qtlibrary
